## 用途

这个 Skill 用于基于已确认的 Zero 低保真稿生成高保真视觉参考图，并完成视觉拆件、素材命名、组件化判断和 Zero 可编辑重建。

## 适用场景

- 低保真稿已经由用户确认，需要进入高保真视觉阶段。
- 需要生成完整高保真参考图，用于确认整体风格、构图、色彩和材质。
- 需要拆分背景、主体、光效、前景装饰，并将文字、按钮、卡片等 UI 元素在 Zero 中重建为可编辑图层。

## 输入

- 用户已确认的 Zero 低保真稿。
- 低保真稿中的页面结构、槽位、文案和交互关系。
- 用户提供或确认的视觉风格、主题、品牌要求和生成提示词。
- 按需使用的 Zero 标准组件信息。

## 输出

- 高保真完整参考图。
- 视觉拆件清单。
- 拆件素材命名建议。
- Zero 高保真可编辑重建要求。
- Zero 高保真可编辑链接、视觉拆件素材和组件拆解清单。

## 边界和风险

- 高保真图只能作为参考层，不能作为最终唯一交付物。
- 所有文字、按钮、卡片、弹窗、可点击入口必须在 Zero 中重建为可编辑图层或组件。
- 已有标准组件不得从高保真 JPG 中抠取，也不得用图像模型重新生成。
- 无法可靠拆出的元素必须明确标记处理方式。
- 视觉风格、品牌规范、奖励素材和活动利益点需要业务方或设计负责人审核。
